# 🏡 Airbnb Clone - SwiftUI iOS App

A modern Airbnb-inspired iOS application built using **SwiftUI**, showcasing property listings, detailed views, map integration, and search functionality.
The app demonstrates strong fundamentals in **UI design, navigation, and state management** using a clean and scalable architecture.

---

## 🚀 Features

* 🏠 Browse property listings with dummy data
* 🔍 Search functionality for destinations
* 📍 Map integration using MapKit (location-based focus)
* 🧭 Smooth navigation between list and detail views
* 🏡 Detailed property screen with rich UI
* 📱 Tab-based navigation for better UX

---

## 🛠️ Tech Stack

* **SwiftUI** – Declarative UI framework
* **MapKit** – Location and map rendering
* **MVVM Architecture** – Scalable code structure
* **Swift** – Core programming language

---

## 🧠 Architecture

The project follows the **MVVM (Model-View-ViewModel)** pattern:

* **Model** → Represents property/listing data
* **ViewModel** → Manages business logic and state
* **View** → SwiftUI views for UI rendering

---

## 📍 Map Integration

* Displays location using **latitude & longitude**
* Focuses map dynamically on selected destination (e.g., Mumbai)
* Built using **MapKit + SwiftUI Map**
